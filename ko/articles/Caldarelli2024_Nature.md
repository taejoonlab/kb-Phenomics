## Background
- Stem cell-derived embryo models이 발생 기전 연구에 혁신, self-organization 원리 이해 부족으로 재현성 제한
- AI 기반 선별이 정상 발생 모델 식별, self-organization 핵심 인자 규명
- Live imaging + deep learning이 초기 발생 동적 과정 포착
- Mouse post-implantation 모델이 인간 발생·질병 연구에 translational potential

## Key Experiment Methods
1. **AI 기반 embryo model 선별**: live imaging + CNN으로 normal/abnormal ETiX-embryo 분류, 발달 궤적 예측
2. **Self-organization 원리 규명**: 세포 수·공간 배열·신호전달 경로가 self-organization에 미치는 영향 정량화
3. **Perturbation 분석**: 초기 조건 변형(세포 수, 배지 조성, 억제제 처리)으로 self-organization 인과관계 검증
4. **Cross-species 비교**: mouse embryo model과 인간 발생 비교, 보존적 self-organization 원리 식별
5. **최적화 파라미터**: CNN 아키텍처, 이미징 시간대, perturbation 조건, self-organization metric 정의

## Results
- AI가 정상 발생 embryo model 고정밀도 선별, 발달 궤적 조기 예측
- 세포 수·공간 배열이 self-organization 성공 핵심 인자 규명
- Perturbation 실험으로 self-organization 인과관계 검증
- Mouse-인간 간 보존적 self-organization 원리 식별
- AI 기반 선별이 stem cell-derived 발생 모델 재현성 향상

## Perspective
- AI + stem cell-derived embryo models이 발생생물학 연구 패러다임 변화
- Self-organization 원리 규명이 재생의학·장기 형성 연구에 기여
- AI 기반 품질 관리가 임상 적용 가능한 embryo model 생산 표준화
- 인간 embryo model 확장 시 질병 모델링·신약 개발 가속
- Self-organization 이해가 synthetic morphology·인공 장기 개발로 이어짐
