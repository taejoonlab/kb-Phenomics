# The Xenopus phenotype ontology: bridging genotype to phenotype

## Background
- Xenopus 발생·질병 모델 연구에서 표현형 기술이 비표준화, 종 간 비교·데이터 통합 어려움
- Phenotype ontology가 유전자형-표현형 관계 체계적 연결, 고속 스크리닝 결과 해석 표준화
- Xenbase 등 모델 생물 데이터베이스에 통합 가능한 ontology 필요
- CRISPR 스크리닝·화학 유전학에서 대규모 표현형 데이터의 의미론적 주석(annotation) 필요

## Key Experiment Methods
1. **Xenopus phenotype ontology 개발**: 발생 단계별·장기별 표현형 용어 체계화, mammalian phenotype ontology(MP)와 매핑
2. **Xenbase 통합**: Xenbase 데이터베이스에 ontology 기반 표현형 주석, 유전자-표현형 연관 검색 기능
3. **종 간 비교**: Xenopus 표현형과 human disease phenotype, zebrafish, mouse 표현형 cross-reference
4. **CRISPR 스크리닝 적용**: 대규모 유전자 녹다운/녹아웃 스크리닝 결과의 ontology 기반 주석, 표현형 프로파일 비교
5. **최적화 파라미터**: ontology 용어 granularity, 매핑 정확도, 주석 일관성, cross-species homology 기준

## Results
- Xenopus 발생·질병 모델 표현형 표준화 ontology 정립
- Xenbase 통합으로 유전자-표현형 연관 검색·데이터 질의 기능 향상
- 종 간 표현형 비교로 human disease gene의 Xenopus ortholog 기능 예측
- CRISPR 스크리닝 결과의 ontology 기반 주석으로 고속 표현형 분류 가능
- Occurrence 데이터의 의미론적 통합으로 메타분석·시스템 생물학 연구 지원

## Perspective
- Phenotype ontology가 Xenopus 기능 유전체학의 표준 데이터 인프라로 자리잡음
- 종 간 ontology 매핑이 comparative developmental biology·질병 모델링 가속
- AI 기반 표현형 자동 주석과 결합하여 고속 스크리닝 파이프라인 완성
- 다른 모델 생물 ontology와 통합으로 범용 발생 phenotype ontology 발전
- 임상 유전체 변이 해석에 Xenopus 표현형 데이터 기여
